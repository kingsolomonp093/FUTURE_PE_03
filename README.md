# AI SEO Blog & Content Cluster Generator

## Project Overview
This project demonstrates how Prompt Engineering can be used to generate SEO-optimized content clusters for business websites.

## Business Chosen
Nillo House of Furniture, Coimbatore

## Problem Solved
Many businesses publish random blog content without a structured SEO strategy. This project creates a reusable prompt workflow for generating pillar blogs, supporting blogs, and local SEO content.

## Prompt Engineering Process
1. Keyword Research
2. Search Intent Analysis
3. Pillar Blog Prompt
4. Supporting Blog Prompt
5. Local SEO Prompt
6. Content Cluster Prompt

## Deliverables
- 1 Pillar Blog
- 5 Supporting Blogs
- Keyword Strategy
- Content Cluster Map
- Local SEO Framework

## Tools Used
- ChatGPT
- Google Search
- Google Trends
- AnswerThePublic

## Outcomes
Generated a complete SEO content pack that can be used by local businesses and digital marketing agencies.
